Macros

No variables detected. Type {{name}} in your prompt to add one.

Shows the most recent resolved values for {{name}}, {{list::...}}, {{random::...}}, and {{roll:...}}.
No macro state yet. Generate or chat once to populate.
{{variable}}

Named variable — replaced with the value set in the Variables panel.

Example: {{topic}} → the value of the "topic" variable

{{random::a::b::c}}

Picks one item at random each time the prompt is sent. No item limit.

Example: {{random::history::science::math}}

{{list::a::b::c}}

Iterates through items in order, cycling back to the start. Useful for bulk generation.

Example: {{list::beginner::intermediate::advanced}}

{{roll:NdN+M}}

Rolls dice. Supports standard dice notation. The modifier (+M or -M) is optional.

Examples: {{roll:1d20}}, {{roll:2d6+3}}, {{roll:1d100-5}}

{{// comment}}

Comment macro — stripped from the prompt before it is sent to the API. Useful for notes.

Example: {{// This is a note, not sent}}

Nesting: Macros inside variable values also resolve. For example, if variable topic is set to random::history::science, then {{topic}} will pick one randomly.

Format: NdN or NdN+M or NdN-M
